# Bicis Mini Web Games

This repository contains small interactive web games built with HTML, CSS and JavaScript that I built for practice.

## Games
- Virtual Piano (with JSON + fetch API)
- Memory Game

## Features
- Keyboard interaction
- Sound playback
- Dynamic data loading (JSON)
- Game logic
- JavaScript DOM Manipulation

## How to run
Open `index.html` in your browser.
